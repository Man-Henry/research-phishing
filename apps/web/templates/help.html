Help & FAQ

How to use the Email Detector?

1. Go to Email Detector
2. Paste the email body in the first field
3. (Optional) Paste email headers in the second field
4. Click Analyze Email
5. Review the results and recommendations

What does "PHISHING" mean?

Phishing is a fraudulent attempt to steal your information (passwords, credit cards, etc.) by disguising an email as legitimate.

What are the email features analyzed?

The system checks:

  • Email authentication (SPF, DKIM, DMARC)
  • URL analysis and suspicious links
  • Content patterns and keywords
  • Sender reputation indicators
  • Language and formatting anomalies

What's a good confidence score?

>80% Very likely phishing or legitimate
50-80% Moderate confidence
<50% Low confidence, review manually

How to use the File Analyzer?

1. Go to File Analyzer
2. Click on the file input or drag & drop
3. Select a file from your computer
4. Results appear automatically
5. Review file hashes and analysis

What file types are supported?

Supported formats: .exe, .dll, .bin, .elf, .zip, .rar, .pdf, .doc, .docx
Maximum file size: 50MB

What are file hashes used for?

MD5, SHA1, SHA256 are unique fingerprints of your file.
You can use these hashes to:

  • Look up the file on VirusTotal.com
  • Verify file authenticity
  • Track file origins

What is file entropy?

High entropy (7-8) = Compressed or encrypted (more suspicious)
Low entropy (0-3) = Plain text or simple data (less suspicious)
This helps detect malware that's compressed to hide its true nature.

How accurate is this system?

Accuracy depends on the model used:

  • Email detection: 50-95% (varies with data quality)
  • File analysis: 60-98% (varies with file types)
Always use this as a tool, not the sole decision-maker.

Is my data safe?

In this demonstration version: Uploaded files are analyzed and deleted immediately.
For production use, implement proper security controls and compliance measures.

What should I do if phishing is detected?

1. Do NOT click any links in the email
2. Do NOT download attachments
3. Do NOT reply with personal information
4. Delete the email
5. Report to your IT security team or email provider

What should I do if malware is detected?

1. Do NOT execute the file
2. Quarantine the file immediately
3. Report to your IT security team
4. Run a full system scan with antivirus
5. Review what could have been compromised

Tip: For best results, combine this tool with other security measures like email filters, antivirus software, and user awareness training.
